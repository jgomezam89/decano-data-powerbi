# Decano Data - Dashboard Power BI

## Descripción

Decano Data es un proyecto de análisis de datos realizado con Power BI para analizar el rendimiento del Recreativo de Huelva dentro del Grupo 4 de Segunda Federación.

El objetivo del proyecto es transformar datos deportivos en información visual y útil mediante KPIs, limpieza de datos, modelado y visualización.

## Herramientas utilizadas

- Power BI Desktop
- Power Query
- DAX
- CSV
- Limpieza de datos
- Visualización de datos

## KPIs analizados

- Posición
- Puntos
- Partidos jugados
- Puntos por partido
- Goles a favor
- Goles en contra
- Diferencia de goles
- Nivel ofensivo
- Nivel defensivo
- Zona de clasificación

## Proceso realizado

1. Recopilación de datos deportivos.
2. Validación manual de la clasificación.
3. Limpieza y transformación de datos en Power Query.
4. Creación de medidas y columnas calculadas.
5. Diseño del dashboard en Power BI.
6. Análisis del rendimiento del Recreativo de Huelva.
